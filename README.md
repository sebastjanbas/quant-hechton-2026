# Quant Hackathon App

A personal finance and portfolio management web app built for the 2026 Quant Hackathon.

## Features

- **Dashboard** — portfolio overview with allocation and performance charts
- **Portfolio** — track holdings, sector breakdown, and live market data via WebSocket
- **Budget** — income/expense tracking with survival calculator
- **Risk Simulation** — future asset value prediction across different scenarios
- **Auth** — secure sign-in with Better Auth

## Stack

- Next.js 16, React 19, TypeScript
- PostgreSQL (`pg`)
- Tailwind CSS v4, shadcn/ui, Recharts
- Socket.io for real-time market data

## Getting Started

```bash
npm install
npm run dev
```
